Write a function that calculates the least common multiple of its arguments; each argument is assumed to be a non-negative integer.


___soln.___
```
var gcd=function(a,b)
{
    var t;
    while(b!==0)
    {
        t=b;
        b=a%b;
        a=t;
    }
    return a;
};
```
